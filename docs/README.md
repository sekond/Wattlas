# docs/ — Wattlas documentation

Supporting documentation for the staged Wattlas expansions. The primary runbook
entry point (`RUN.md`), the human-facing overview (`README.md`), and the Claude
Code instructions (`CLAUDE.md`) stay at the repo root.

- **runbooks/** — gated execution protocols Claude Code follows one step per turn:
  `RUN_V2.md` (dashboard expansion), `RUN_V3.md` (wasted-wind / DE north–south),
  `RUN_V4.md` (France nuclear), `RUN_V10.md` (Value-Layer review & plan).
- **roadmaps/** — the plans the runbooks execute: `ROADMAP_V2.md`,
  `ROADMAP_V10.md`, and `wattlas-consolidated-roadmap.md` (the v10 source synthesis).
- **slices/** — per-slice specs and build packages: `SLICE_DE_WASTED_WIND.md`,
  `SLICE_FR_NUCLEAR.md`, `WASTED_WIND_SLICE_README.md`.
- **SOURCES.md** — the European data-source catalogue (ENTSO-E, MaStR, SMARD,
  netztransparenz, RTE, …).
- **CHANGELOG.md** — project changelog.
